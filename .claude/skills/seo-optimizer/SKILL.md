# SEO Optimizer for Developer Portfolios

A comprehensive SEO optimization skill specifically designed for developer portfolios to enhance search engine visibility and attract recruiters through strategic optimization techniques.

## Overview

This skill provides advanced SEO analysis and optimization capabilities tailored specifically for developer portfolios. It combines technical SEO best practices with recruiter-focused strategies to maximize visibility in job searches and technical searches.

## Core Capabilities

### 1. Technical SEO Analysis
- **Core Web Vitals Audits**: Performance analysis using Lighthouse integration
- **Meta Tags Optimization**: Automatic generation and optimization of title tags, meta descriptions
- **Structured Data Implementation**: JSON-LD schema for projects, skills, and experience
- **Crawlability Analysis**: robots.txt optimization and sitemap generation
- **Mobile-First Optimization**: Responsive design checks and mobile performance testing
- **Image Optimization**: Alt text generation and loading performance improvements

### 2. Developer-Focused Keyword Strategy
- **Technical Keywords**: Optimize for technology-specific searches (React Developer, Python Engineer)
- **Location-Based SEO**: Target local job markets with geographic keywords
- **Experience Level Targeting**: Junior, Mid-level, Senior developer keyword optimization
- **Industry-Specific Terms**: Fintech, Healthtech, E-commerce specialization keywords
- **Long-Tail Keywords**: Niche technology combinations and project-specific terms

### 3. Recruiter Optimization Strategies
- **ATS-Friendly Content**: Applicant Tracking System optimization techniques
- **Job Title Matching**: Align portfolio content with common recruiter search patterns
- **Skills Taxonomy**: Organize technical skills using industry-standard terminology
- **Experience Highlighting**: Optimize project descriptions for recruiter scanning
- **Call-to-Action Optimization**: Improve contact and engagement rates

### 4. Content SEO Enhancement
- **Header Structure Optimization**: Proper H1-H6 hierarchy for better crawling
- **Internal Linking Strategy**: Strategic linking between projects and skills
- **Content Gap Analysis**: Identify missing keywords and content opportunities
- **Readability Optimization**: Improve content accessibility for both humans and bots
- **Featured Snippet Targeting**: Optimize content for Google's featured snippets
